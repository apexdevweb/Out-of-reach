🧠 Architecture MVVM (Model-View-ViewModel)
➤ Principe

MVVM est une évolution de MVC adaptée aux interfaces modernes.

Elle sépare :

Model → données (API, state)
View → interface utilisateur (HTML)
ViewModel → logique entre les deux

👉 Le ViewModel fait le lien automatiquement (binding).
